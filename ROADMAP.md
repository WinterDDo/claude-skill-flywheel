# Roadmap

Ideas, not promises. The project is small on purpose, and most of these keep it that way.

- A short recorded demo of the loop running, to replace the static one in the README.
- Install notes for Windows and PowerShell.
- A few example distilled principles, so a fresh install has something to read on day one.
- A cleaner summary view for `/flywheel:flywheel-status`.
- A worked example of using the portable prompt outside Claude Code.

If one of these matters to you, the issues are open, and a [distilled principle](CONTRIBUTING.md) is the most welcome contribution of all.
