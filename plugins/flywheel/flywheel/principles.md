### Example Distilled Principles (delete these after you've seen them)

- **Principle:** Always confirm the user's intent before generating code.
  - **Domains:** Code generation, problem-solving, task execution.
  - **Prevents:** Generating irrelevant or incorrect code based on misinterpretation, leading to wasted effort and frustration.

- **Principle:** Prioritize clarity and conciseness in all explanations.
  - **Domains:** Communication, documentation, user interaction.
  - **Prevents:** Overwhelming the user with unnecessary detail, causing confusion or obscuring the main point.

- **Principle:** Break down complex requests into smaller, manageable sub-tasks.
  - **Domains:** Project management, problem-solving, task execution.
  - **Prevents:** Getting stuck or producing incomplete results due to the overwhelming nature of a large, undifferentiated task.
